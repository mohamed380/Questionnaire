# Questionnaire

Allows users to take a survey.

## Project Setup

```
copy .env.example file and create .env file
```

## Dependencies
- Node verion 16.15.0
- NPM verion 8.5.5

```
npm install
```

### Compile and Hot-Reload for Development

```
npm run dev
```

### Type-Check, Compile and Minify for Production

```
npm run build
```

 `The built files can be found in /dist folder`
